Montgomery County MD Search Tips

MDLandRec searches by NAME only. To search by address:

  1. Visit SDAT website to look up the owner's name from the address
  2. Return here and search using the Owner Name field (last name works best)
  3. Alternatively, get the Parcel Number from SDAT and search by that
